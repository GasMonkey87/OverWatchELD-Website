Proper Use

How to use OverWatch ELD correctly

1

Start ATS first

Open American Truck Simulator, load into your profile, then connect the ELD to telemetry. This helps the ELD detect truck, trailer, location, cargo, mileage, and duty changes.

2

Sign in and link your VTC

Log into the ELD, verify your driver name, and make sure your Discord/VTC link is correct so BOLs, alerts, scores, maintenance, and dispatch records use the right driver.

3

Run pre-trip inspections

Before driving, complete the pre-trip/vehicle inspection. Report defects instead of ignoring them so managers and mechanics can track issues.

4

Use dispatch and BOL tools

Import or accept assigned loads, review route/cargo/truck details, then generate BOL paperwork when needed. Make sure pickup and delivery data is correct.

5

Report problems with details

When something breaks, include your ELD version, ATS version, what you clicked, screenshots, Discord server/VTC name, and any error text.

FAQ

Frequently asked questions

How do I install OverWatch ELD?

Go to Downloads, download the latest Windows EXE or installer, place it in a folder you control, then run it. If Windows warns you about an unknown app, verify you downloaded it from the official OverWatch ELD website before continuing.

Why is ATS telemetry not connecting?
  • Start ATS before pressing connect.
  • Load into your ATS profile first.
  • Make sure the telemetry plugin is installed and not blocked.
  • Restart the ELD after ATS is fully loaded.
  • If the truck still shows unknown data, include screenshots and your ATS version in a support request.
Why does my truck, plate, location, or cargo show unknown?

The ELD can only show data that ATS telemetry provides. Load fully into the truck, attach the trailer/load, then reconnect telemetry. If you use modded trucks/trailers, some data may appear differently depending on the mod.

How do I add the Discord bot to my server?

Open the Downloads page and click “Add Bot to Server.” You must have permission to add apps or manage the Discord server. After adding the bot, configure your VTC channels and link users.

Who can access Dispatch?

Dispatch tools are intended for Admins, Managers, and Dispatchers. Regular drivers should receive or import loads, but should not manage company-wide dispatch unless assigned the proper role.

Why did my BOL or Discord notification not send?
  • Confirm your VTC/Discord link is active.
  • Make sure the bot is still in your server.
  • Verify the correct BOL, dispatch, logs, or maintenance channel is configured.
  • Check that the ELD has internet access and your load has valid pickup/delivery details.
How should drivers use logs and duty status?

Drivers should connect telemetry, keep duty status accurate, complete inspections before driving, review logs before certifying, and avoid editing records unless they are correcting a real mistake.

How do inspections and maintenance tickets work?

Pre-trip, post-trip, and vehicle inspections help detect defects. If an issue is found, create or submit a maintenance request so management/mechanics can review, repair, and track the vehicle history.

What should I include in a bug report?
  • ELD version and ATS version.
  • What screen or button you used.
  • Exact error message or screenshot.
  • Whether ATS was running and loaded into your profile.
  • Your Discord server/VTC name if the issue involves bot or portal features.
Can I use OverWatch ELD without a VTC?

Yes. You can use driver tools individually. VTC features such as Discord alerts, dispatch, fleet analytics, applications, and shared dashboards work best after the bot and server link are configured.

Where do I get the latest version?

Use the Downloads page. It checks GitHub releases and points to the newest release asset when available.

What if I cannot submit the support form?

Use the “Open Email Manually” button or email support directly from your email app. Include the same details requested in the form.
